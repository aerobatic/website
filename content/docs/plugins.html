Plugins

Plugins are declared in your aerobatic.yml file and allow you to augment your static site with additional capabilities. Visit the plugin configuration guide to learn more about how they work. Then come back here to explore all the available plugins.

redirect

Define HTTP redirect rules

password-protect

Apply password protection to your website

custom-errors

Custom pages for 404 and other error codes

http-proxy

Proxy http calls to remote endpoints

http-headers

Define your own HTTP headers and add security headers

s3-proxy

Serve files directly from your S3 bucket

cors

Make certain resources available via CORS

client-config

Expose config settings to client JavaScript

webpage

Control webpage serving and pushState

basic-auth

HTTP Basic Authentication

health-check

Dedicated endpoint for your health monitoring service
